## 🎯 Project Objective

This project is created to help students learn how to build a professional sales analytics dashboard in Power BI using real-world retail data from Adidas.

## 📊 What This Dashboard Shows

- Overall sales performance and key KPIs (Revenue, Units Sold, Profit)
- Sales trends over time with year-over-year comparison
- Regional performance across different markets
- Product category and item-level analysis
- Top-performing products and underperforming segments
- Interactive filters for deep dive analysis

## 🧑‍🏫 Learning Objectives (For Students)

By completing and exploring this project, you will learn:

1. **Data Modeling** – Building relationships between sales, products, and region tables
2. **DAX Measures** – Creating calculations for Total Sales, YoY Growth, Profit Margin, etc.
3. **Interactive Visuals** – Using bar charts, line charts, maps, cards, and slicers effectively
4. **Dashboard Layout & Design** – Applying professional themes and best UI practices
5. **Sales Analytics Thinking** – Turning raw sales data into actionable business insights
6. **Power Query** – Cleaning and transforming retail datasets

## 🛠️ Technologies Used

- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
- Custom Theme (CY25SU10)

## 📁 Project Files

- `Adidas Sales Dashboard.pbix` ← Main Power BI file
- Embedded Adidas logo and icons
- Pre-written DAX queries

## 🚀 How to Use This Project (Step-by-Step Guide for Students)

1. Download the `Adidas Sales Dashboard.pbix` file
2. Open it using **Power BI Desktop** (Free version works perfectly)
3. Explore each report page and interact with the slicers
4. Switch to **Model** view to understand table relationships
5. Go to **Data** view to explore the cleaned dataset
6. Review the DAX measures under the **DAXQueries** folder
7. Try modifying existing visuals or creating new ones

## 📈 Dashboard Pages Included

- **Overview** – Executive summary with key performance indicators
- **Sales Trends** – Time series analysis and growth patterns
- **Regional Analysis** – Performance by country/region
- **Product Performance** – Category-wise and top products breakdown
- **Profitability** – Margin and profit insights

## 💡 Student Challenges / Assignments

- Create a new measure for "Average Order Value"
- Build a custom tooltip for product cards
- Add a "What-If" parameter for target sales analysis
- Redesign one page using your own color scheme
- Prepare a 2-minute presentation explaining the insights from this dashboard
